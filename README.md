# Бот-SST
 
SST бот для преобразования голосового сообщения в текст.

 
## Описание

Бот умеет преобразоввывать голосовое сообщение в текст
 
 

## Инструкции по использованию
Введи команду
 
  - /sst

  - Отправить голосовое сообщение
